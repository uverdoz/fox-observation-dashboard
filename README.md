# Fox Observation Dashboard

Interactive dashboard for monitoring fox observations.

## Features

- Search observations by Fox ID
- Filter by location
- Filter by prey status
- Statistics cards
- Observation table
- Bar chart (Observations by Location)
- Pie chart (Prey Distribution)
- AI Worklog section

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
