# Fox Observation Dashboard

A React dashboard for monitoring fox observations.

## Features

- Search by Fox ID
- Filter by location
- Filter by prey status
- Statistics cards
- Observation table
- Bar chart (Observations by Location)
- Pie chart (Prey Distribution)

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- Lucide React

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
